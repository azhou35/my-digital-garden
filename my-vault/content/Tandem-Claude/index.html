    # CORE FRAMEWORKS

    # 1. PROBLEM-DIAGNOSIS-SOLUTION (PDS)

    • Problem: Clearly define the issue (metrics, impact, scope)
      • Example: “7-day PA delays causing 15% prescription abandonment”
    • Diagnosis: Root cause analysis using data
      • Example: “Manual form processing, missing data, error-prone workflow”
    • Solution: Tandem’s AI-driven approach with quantifiable benefits
      • Example: “AI automation reduces processing to 2 days, cuts abandonment by 50%”

    # 2. PROCESS-PEOPLE-TECHNOLOGY (PPT)

    • Process: Map current workflow, identify bottlenecks
      • Steps: Prescription ‚Üí PA submission ‚Üí Review ‚Üí Decision ‚Üí Fulfillment
      • Bottlenecks: Form completion, information gathering, manual reviews
    • People: Stakeholder analysis (who’s involved, pain points)
      • Stakeholders: Patients, doctors, staff, insurers, pharmacies
      • Pain Points: Time burden, administrative costs, patient care delays
    • Technology: Tandem’s AI solution addressing specific gaps
      • Features: Auto-form completion, error detection, approval prediction

    # 3. MARKET OPPORTUNITY ANALYSIS

    • Size: Quantify TAM/SAM/SOM
      • Example: “$73B oncology PA market with 40% denial rate”
    • Competition: Current landscape assessment
      • Example: “Manual PA services with 7-day turnaround vs. Tandem’s 2-day AI approach”
    • Strategy: Differentiated approach to win market share
      • Example: “Partner with specialty clinics, target high-denial therapeutic areas”

    # 4. HYPOTHESIS-DRIVEN APPROACH

    • Hypothesis: Initial assumption based on available data
      • Example: “Manual data entry causes 70% of PA delays”
    • Test: How to validate the hypothesis
      • Example: “Analyze time spent by step in current PA workflow”
    • Analyze: Data-driven evaluation
      • Example: “Form completion takes 4 of 7 days, AI reduces to 1 day”
    • Recommend: Clear, actionable next steps
      • Example: “Implement AI form completion to achieve 2-day turnaround”

    # VALUE PROPOSITION ELEMENTS

    • Speed: Faster PA turnaround (7‚Üí2 days)
    • Accuracy: Reduced denial rates (30%‚Üí10%)
    • Affordability: Patient assistance when insurance denies
    • ROI: Provider time savings, reduced abandonment, higher fill rates
