---
title: "planes of existence"
---

gpu node exists in several planes at once

| Plane                                | Core question                         |
| ------------------------------------ | ------------------------------------- |
| **Control plane**                    | _What should exist?_                  |
| **Kubernetes / orchestration plane** | _What should be running where?_       |
| **Management plane**                 | _Can we see / control the machine?_   |
| **Data plane**                       | _Can data actually flow fast enough?_ |
